IDEA 常用Java自定义模板快捷输入

2022/10/27 posted in  IDEA
Tags:  #生产力工具

注意部分冲突，可去掉相关勾选

logg

# logg
# 日志快速创建
# 编辑变量 NAME 表达式 className()
# 适用于Java:声明

# 模板内容
private static final org.slf4j.Logger LOG = org.slf4j.LoggerFactory.getLogger($NAME$.class);

logi

# logi
# LOG.info("");
# 适用于Java:表达式


# 模板内容
LOG.info("$END$");

logw

# logw
# LOG.warn();
# 适用于Java:表达式

# 模板内容
LOG.warn($END$);

loge

# loge
# LOG.error();
# 适用于Java:表达式

# 模板内容
LOG.error($END$);
